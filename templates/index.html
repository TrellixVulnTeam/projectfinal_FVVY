What is a resistor ?

A resistor is a component that is specifically disegned to have a certain amount of resistance.

The principal applications of resistors are:

  • To limit current
  • Divide voltage o=and/or current
  • In some cases generate heat

What is a voltage divider ?

A voltage divider is a simple circuit which turns a large voltage into a smaller one. Using at least two series resistors and an input voltage, we can create an output voltage that is a fraction of the input.


  • Series resistors divide voltage
  • At least two resistors connected in serie divide current

What is a current divider ?

A current divider is a simple circuit which turns a large current into a smaller one. Using at least two parallel resistors and an input current, we can create an output current that is a fraction of the input.

  • Parallel resistors divide current
  • At least two reistors connected in parallel divide current
